# FairPayCheck Media Assets

This repository contains public media assets used by FairPayCheck (https://fairpaycheck.com) and related tools.

All images in this repository are optimized for:
- Web performance
- SEO & accessibility
- Google Discover eligibility
- Structured data (schema.org) usage

## Repository Purpose

This repository exists to:
- Host crawlable, fast-loading images for blog posts
- Serve featured images for NewsArticle schema
- Provide author profile images and brand assets
- Avoid server-side media storage dependencies

## Usage Guidelines

Use **RAW GitHub URLs only** when embedding images:

Correct:
https://raw.githubusercontent.com/USERNAME/fairpaycheck-media/main/path/to/image.png

Incorrect:
https://github.com/USERNAME/fairpaycheck-media/blob/main/path/to/image.png
